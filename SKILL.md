# 全国多版本英语试卷智能生成器（真题对标版）

## 一、适用范围与核心能力
### 覆盖学段
- 小学：1-6 年级（区分低段1-3年级、高段4-6年级）
- 初中：7-9 年级、中考/模拟考

### 支持教材版本
人教PEP（小学）、人教新目标Go for it!（初中）、牛津译林版、外研版、沪教牛津版、苏教版、冀教版、北师大版，共8大主流版本

### 支持考试类型
单元测试、月考、期中、期末、小升初、中考、专项模拟考

### 三大组卷模式
1. **真题复刻**：基于指定地区/年份真题，1:1还原题型结构、分值占比、考点分布与命题风格
2. **AI创新**：结合教材考点生成新颖场景题目，支持跨学科融合与真实生活情境命题
3. **分层定制**：基础层/提升层/拔尖层三级难度，适配不同水平学生

## 二、前置依赖说明
1. **基础模式**：无需外部服务，AI基于内置命题规范直接生成，适合快速出卷
2. **真题增强模式（推荐）**：对接本地 K12 英语真题 MCP 服务，实现三项核心能力
   - 按地区、年份、题型抽取真实真题样本，复刻命题逻辑与干扰项设置
   - 统计历年考点分值占比，精准匹配考纲要求
   - 生成后自动原创性查重，相似度＞20%自动重写
3. **导出模式**：调用配套脚本生成可打印 PDF / 可编辑 Word / 结构化 JSON

## 三、标准执行流程
### 步骤1：参数确认与校验
主动向用户确认以下核心参数，缺失项逐一询问，禁止无依据默认：
| 参数项 | 可选值/说明 | 默认值 |
|--------|-------------|--------|
| 教材版本 | 人教PEP / 人教新目标 / 牛津译林版 / 外研版 / 沪教牛津版 / 苏教 / 冀教 / 北师大版 | 无默认，必须确认 |
| 学段年级 | 小学1-6年级 / 初中7-9年级 / 中考 | 无默认，必须确认 |
| 考试类型 | 单元测试 / 月考 / 期中 / 期末 / 小升初 / 中考模拟 | 无默认，必须确认 |
| 试卷时长 | 可自定义 | 小学60分钟 / 初中90分钟 / 中考120分钟 |
| 总分值 | 可自定义 | 小学100分 / 初中120分 / 中考120/150分 |
| 组卷模式 | 真题复刻 / AI创新 / 分层定制 | AI创新 |
| 分层级别（分层定制模式必填） | 基础层 / 提升层 / 拔尖层 | 无默认 |
| 真题范围（真题复刻模式必填） | 指定省份/城市、年份范围 | 近3年全国通用真题 |
| 交付内容 | 试卷正文 / 答题卡 / 答案解析 / 考点分析报告 | 全部包含 |

### 步骤2：标准模板匹配
根据用户参数，从「四、试卷标准规范库」中匹配对应学段、版本、考试类型的题型结构、分值占比、难度梯度模板，严格按照模板生成，禁止随意增减题型、调整分值比例。

### 步骤3：按组卷模式生成题目
#### 模式A：真题复刻模式
1. 调用本地 MCP 接口 `/mcp/sample_questions`，按题型抽取 3-5 道指定范围的真题作为风格样本
2. 调用 MCP 接口 `/mcp/knowledge_weight`，获取对应考试的考点分值占比，确保考点覆盖度匹配
3. 严格复刻真题的设问句式、选项长度、干扰项逻辑、材料篇幅，保留考点不变，替换场景与数据
4. 生成完成后调用 `/mcp/check_duplicate` 做原创性校验，相似度超标自动重写
5. 每道题标注真题来源参考（如「命题参考：2024年深圳五年级期末真题」）

#### 模式B：AI创新模式
1. 严格限定对应教材版本、年级的词汇、语法、话题范围，杜绝超纲
2. 优先选用真实生活场景、跨学科融合素材（英语+文化/科学/地理/历史）
3. 题型结构与分值严格遵循标准模板，命题角度新颖但不偏离考纲要求
4. 干扰项设置符合真题逻辑：概念混淆、常见错误、审题陷阱，杜绝无效选项
5. 阅读材料篇幅、信息密度与同级别真题保持一致

#### 模式C：分层定制模式
根据分层级别调整难度与考点侧重，题型结构保持标准不变：
- **基础层（60分合格线）**：侧重词汇拼写、基础句型、简单细节阅读，基础题占比≥80%，无超纲拓展
- **提升层（80分良好线）**：侧重语法综合运用、长难句理解、常规写作，中档题占比≥40%，少量拓展
- **拔尖层（100分满分线）**：侧重跨学科融合、推理判断、创新应用，难题占比≥20%，区分度强

### 步骤4：标准化排版
严格按照「五、输出格式规范」生成完整交付内容，包含：
1. 标准卷头与答题须知
2. 听力部分 + 笔试部分，按官方题型顺序排列
3. 每题标注分值，大题标注总分
4. 独立标准答题卡
5. 独立标准答案与逐题解析
6. 考点覆盖度与难度分析报告

### 步骤5：合规性自动校验
生成完成后自动校验以下指标，偏差超过 5% 自动调整：
1. 总分值是否与设定一致
2. 各题型分值占比是否符合标准模板
3. 难度梯度比例是否符合对应考试要求
4. 考点是否覆盖对应教材单元核心知识点
5. 是否存在超纲词汇与语法点

### 步骤6：导出交付
调用配套脚本生成对应格式文件：
- PDF 格式：标准 A4 可打印，适合考场直接使用
- Word 格式：可二次编辑，适合教师调整修改
- JSON 格式：结构化题目数据，适配在线考试系统

### 4.2 小学阶段标准题型与分值（满分100分基准）
#### 听力部分（40分）
| 题型 | 分值占比 | 核心考点与命题逻辑 |
|------|----------|--------------------|
| 听音选图/选词 | 20% | 考查核心词汇（颜色、动物、数字、职业）音义对应；低段以图文匹配为主，高段逐步过渡到纯文字选项 |
| 听句子选答语 | 10% | 考查日常交际句型（询问时间、天气、喜好）语境运用；干扰项设置为常见答非所问逻辑 |
| 听短文填空/判断 | 10% | 考查语篇理解与单词拼写；填写听力材料中原词，拼写错误按0.5分/处扣分；低段为判断正误，高段升级为补全句子 |

#### 笔试部分（60分）
| 题型 | 分值占比 | 核心考点与命题逻辑 |
|------|----------|--------------------|
| 单词分类/选同类词 | 5% | 考查词汇语义归类（水果类、职业类、场所类等）；干扰项设置为易混淆范畴词汇 |
| 单项选择 | 20% | 考查词汇辨析、固定搭配、基础语法（be动词、情态动词、时态基础）、句型结构；干扰项覆盖常见语法错误 |
| 情景交际 | 5% | 考查日常交际用语得体运用；设置真实生活场景（问路、看病、购物），选择最恰当回答 |
| 句子翻译/英汉互译 | 10% | 考查核心短语、固定搭配、简单句型转换；要求单词拼写准确 |
| 阅读理解 | 25% | 低段以图文匹配、信息提取为主；高段增加短文理解，考查细节判断与简单推理；2-3篇材料 |
| 写作表达 | 15% | 低段：抄写句子、仿写短语；高段：看图写话、简单书信/通知；评分维度：内容完整性、语言准确性、书写规范性 |

### 4.3 初中阶段标准题型与分值（满分120分基准）
#### 听力部分（30分）
| 题型 | 分值占比 | 核心考点与命题逻辑 |
|------|----------|--------------------|
| 短对话理解 | 10% | 考查日常场景对话细节（职业、活动、时间、地点）；干扰项设置为听力原文提及但不符合题干的信息 |
| 长对话理解 | 8% | 考查多轮对话信息整合与逻辑推理；设置2-3个递进问题 |
| 短文/独白理解 | 12% | 考查语篇细节抓取与主旨理解；涵盖人物、事件、原因、数字等考点 |

#### 笔试部分（90分）
| 题型 | 分值占比 | 核心考点与命题逻辑 |
|------|----------|--------------------|
| 单项选择 | 15分 | 考查词汇辨析、语法综合、句型搭配；考点覆盖时态、语态、从句、非谓语等核心语法点 |
| 完形填空 | 20分 | 考查语境理解、词义辨析、语法运用、逻辑关系；干扰项设置为词义混淆、时态错误、逻辑不通三类 |
| 阅读理解 | 30分 | 3-4篇材料，涵盖记叙文、说明文、议论文；题型包括细节理解、主旨概括、推理判断、词义猜测 |
| 语法填空 | 10分 | 考查词性转换、时态语态、固定搭配、冠词介词；需结合上下文逻辑填空 |
| 补全对话/任务型阅读 | 5分 | 考查交际连贯性与信息提炼能力；根据上下文补全对话或回答问题 |
| 书面表达 | 10分 | 考查应用文（书信、通知、演讲稿）或议论文写作；评分维度：内容完整、语言准确、逻辑连贯 |

### 4.4 中考/模拟考专项适配
1. **题型扩展**：增加情景交际（5%）、任务型阅读（10%），作文分值提升至15-20分
2. **难度梯度**：简单题40%、中等题40%、难题20%，全面覆盖课标考点
3. **命题趋势**：融入跨学科融合（英语+文化/地理/科学）、真实情境命题（校园活动、社会热点）
4. **阅读升级**：增加非连续性文本阅读、图表信息提取类题型

### 4.5 各教材版本适配规则
| 教材版本 | 命题适配特点 |
|----------|--------------|
| 人教PEP/人教新目标 | 话题贴合教材单元主题，词汇严格限定教材范围，题型常规基础，侧重句型与语法运用 |
| 牛津译林版 | 阅读材料篇幅更长，任务型阅读占比高，词汇量略大，侧重语境综合运用与推理 |
| 外研版 | 话题覆盖更广，完形填空侧重逻辑推理，写作贴近生活应用场景，交际性强 |
| 沪教牛津版 | 整体难度偏高，词汇量大，阅读话题拓展性强，对长难句理解要求高 |
| 苏教版 | 侧重语法细节考查，句型变换题型丰富，基础知识点考察细致 |
| 冀教版/北师大版 | 话题贴近本土文化，阅读材料生活化，注重语言实际运用能力 |

## 五、输出格式与交付标准
### 5.1 试卷正文标准格式
```

# XX 年级英语 XX 考试试卷

（XX 版・真题复刻 / AI 创新 / 分层定制版）

满分：XX 分  考试时长：XX 分钟

## 注意事项

1. 答题前填写好自己的姓名、班级、考号等信息
2. 请将答案正确填写在答题卡上

---

## 第一部分 听力部分（共 XX 分）

### 一、听录音，选出你所听到的内容（共 XX 小题，每小题 X 分，满分 XX 分）

1. A. xxx  B. xxx  C. xxx
2. ...

```

### 5.2 标准答题卡格式
- 卷头预留姓名、班级、考号填写区域
- 客观题：标准四选一填涂方框，题号与选项一一对应
- 主观题：预留对应答题横线与充足作答空间
- 作文题：预留对应字数的方格/横线答题区域
- 页脚标注页码与「请在答题卡上作答，试卷上答题无效」提示

### 5.3 答案与解析格式
1. 按题型顺序排列答案，客观题给出唯一标准答案
2. 每道题标注对应考点（如「考点：一般现在时第三人称单数」）
3. 难题附带详细解题思路与易错点提示
4. 主观题给出参考答案与分层评分标准
5. 作文题给出参考范文与评分细则

### 5.4 配套分析报告
每套试卷附带「试卷分析报告」，包含：
1. 考点覆盖度统计表：各单元/知识点分值占比
2. 难度分布统计图：基础/中档/难题占比
3. 题型分值明细表
4. 命题说明与适用场景建议

### 5.5 导出格式支持
1. **PDF格式**：标准A4排版，宋体，适合直接打印使用
2. **Word格式**：可编辑文档，保留格式与排版，支持教师二次修改
3. **JSON格式**：结构化题目数据，包含题型、题干、选项、答案、考点、难度等字段，适配在线考试系统

## 六、配套脚本与接口说明
### 6.1 PDF/Word导出脚本
调用路径：`scripts/generate_exam_files.py`
- 输入：标准Markdown格式试卷全文
- 输出：试卷正文.pdf、答题卡.pdf、答案解析.pdf、试卷正文.docx
- 排版规则：A4纸张、2cm边距、宋体、14号正文字号、标准行间距

### 6.2 本地真题MCP对接接口
增强模式下调用以下 MCP 接口：
1. 真题样本抽取：`GET /mcp/sample_questions`
   参数：学科、学段、题型、地区、年份、数量
2. 考点权重统计：`GET /mcp/knowledge_weight`
   参数：学科、学段、考试类型、地区
3. 原创性查重：`POST /mcp/check_duplicate`
   参数：题目文本、相似度阈值
4. 难度分布查询：`GET /mcp/difficulty_distribution`
   参数：学科、学段、题型

## 七、边界与使用限制
1. 听力部分仅提供听力材料文本与题目，不生成音频文件
2. 纯AI模式下教材考点为通用核心范围，精准匹配教材单元需对接对应教材知识点库
3. 真题复刻模式需本地部署真题MCP服务并导入对应地区真题数据
4. 禁止用于商业用途，对接第三方题库需遵守对应平台版权规则
5. 生成的题目为AI辅助创作，正式考试使用前建议人工审核校验
