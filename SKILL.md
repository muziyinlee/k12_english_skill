# K12 英语标准试卷生成 Skill

## 一、适用场景
- 小学3-6年级：单元测、月考、期中、期末、小升初英语试卷生成
- 初中7-9年级：单元测、月考、期中、期末、中考模拟英语试卷生成
- 支持教材版本：人教PEP（小学）、人教新目标Go for it!（初中）、牛津译林版、外研版、沪教牛津版
- 输出标准：完全贴合国内公立校考试格式，支持直接打印、考场使用
- 配套产出：试卷正文、标准答题卡、标准答案与解析，可一键导出A4打印PDF

## 二、前置依赖
1. **基础模式**：纯AI生成，无需额外依赖，直接输出标准Markdown格式试卷
2. **增强模式（推荐）**：对接本地K12英语真题MCP服务，实现：
   - 本地真题命题风格1:1复刻
   - 考点分布精准对齐当地考情
   - 原创性查重，避免生成现成原题
3. **PDF导出模式**：本地Python环境安装 `weasyprint` 库，调用配套脚本生成可打印PDF

## 三、核心执行流程
### 步骤1：参数确认与校验
主动向用户确认以下核心参数，缺失项逐一询问，禁止默认猜测：
1. 学段：小学 / 初中
2. 年级：3-6年级（小学）/ 7-9年级（初中）/ 中考
3. 教材版本：人教PEP / 人教新目标 / 牛津译林版 / 外研版 / 沪教牛津版 / 其他
4. 考试类型：单元测 / 月考 / 期中 / 期末 / 小升初 / 中考模拟
5. 满分分值：默认遵循对应考试官方标准（小学100分/初中期中期末120分/中考120/150分）
6. 难度偏好：基础 / 中等 / 拔高，默认遵循对应考试官方难度比例
7. 是否包含听力部分：默认包含
8. 是否需要答题卡、答案解析：默认全部生成

### 步骤2：标准模板匹配
根据用户参数，从「四、试卷标准规范库」中匹配对应的题型、题量、分值、时长、难度分布模板，严格按照模板结构生成试卷，不得随意增减题型、调整分值占比。

### 步骤3：题目生成
#### 基础模式（无本地MCP）
- 严格贴合对应教材版本的词汇、语法、话题范围，不超纲
- 按照难度比例分层命题：基础题考察直接知识点，中档题考察综合运用，难题考察语境理解与拓展
- 选择题干扰项符合真题挖坑逻辑：概念混淆、常见错误、审题陷阱，杜绝一眼可排除的无效选项
- 阅读材料贴合对应学段话题，篇幅、信息密度与真题保持一致

#### 增强模式（对接本地MCP服务）
1. 调用本地MCP接口 `/mcp/sample_questions`，按题型抽取3-5道同年份同地区真题作为风格样本
2. 调用 `/mcp/type_distribution` 获取考点分值占比，对齐考点分布
3. 基于真题样本复刻命题风格、设问方式、选项长度与逻辑
4. 生成完成后调用 `/mcp/check_duplicate` 做原创性校验，相似度＞20%自动重写对应题目

### 步骤4：标准化排版
严格按照「五、输出格式规范」生成Markdown格式试卷，包含：
- 标准卷头（试卷名称、满分、考试时长、答题注意事项）
- 分板块排版（听力部分 → 笔试部分，按官方题型顺序排列）
- 每题标注分值，大题标注总分
- 单独的答题卡页面
- 单独的答案与解析页面

### 步骤5：难度合规校验
生成完成后自动校验整套试卷难度分布，偏差超过5%自动调整：
- 小学阶段：基础题≥70%，中档题≤20%，难题≤10%
- 初中/中考阶段：基础题≥60%，中档题≤30%，难题≤10%

### 步骤6：PDF导出
调用配套脚本 `scripts/pdf_generator.py`，将Markdown试卷转换为标准A4纸可打印PDF，输出三个独立文件：
1. `试卷正文.pdf`：仅题目，用于考场发放
2. `答题卡.pdf`：标准填涂答题卡，含客观题填涂区与主观题答题区域
3. `答案解析.pdf`：标准答案+逐题解析，用于阅卷讲解

## 四、试卷标准规范库
### 4.1 学段核心参数总表
| 学段 | 典型考试 | 满分标准 | 考试时长 | 难度比例（基础:中档:难题） |
|------|----------|----------|----------|----------------------------|
| 小学3-4年级 | 单元测 | 100分 | 40分钟 | 8:1:1 |
| 小学3-4年级 | 期中/期末 | 100分 | 60分钟 | 7:2:1 |
| 小学5-6年级 | 单元测/月考 | 100分 | 45分钟 | 7:2:1 |
| 小学5-6年级 | 期中/期末 | 100分 | 60分钟 | 7:2:1 |
| 小学 | 小升初 | 100/120分 | 90分钟 | 7:2:1 |
| 初中7-8年级 | 单元测/月考 | 100分 | 60分钟 | 7:2:1 |
| 初中7-8年级 | 期中/期末 | 120分 | 90分钟 | 6:3:1 |
| 初中9年级 | 月考/模考 | 120/150分 | 100分钟 | 6:3:1 |
| 初中 | 中考模拟 | 120/150分 | 120分钟 | 6:3:1 |

### 4.2 小学阶段标准题型与分值（以100分为例）
#### 听力部分（30-40分）
1. 听录音选图片/单词：10分（5题，每题2分）
2. 听录音判断对错：10分（5题，每题2分）
3. 听录音选答语/补全对话：10分（5题，每题2分）
4. 听录音填空（5-6年级）：10分（5空，每空2分）

#### 笔试部分（60-70分）
1. 辨音题/词汇匹配：10分
2. 单项选择：15-20分（每题1分）
3. 完形填空（5-6年级）：10分（10题，每题1分）
4. 阅读理解：15-20分（2-3篇，每篇5题）
5. 句型转换/连词成句：5-10分
6. 书面表达：5-10分（3-4年级5句话，5-6年级8-10句话）

### 4.3 初中阶段标准题型与分值（以120分为例）
#### 听力部分（25-30分）
1. 短对话理解：10分（10题，每题1分）
2. 长对话理解：5分（5题，每题1分）
3. 短文/独白理解：10-15分（5-10题，每题1分）

#### 笔试部分（90-95分）
1. 单项选择：15分（15题，每题1分）
2. 完形填空：15分（15题，每题1分）
3. 阅读理解：30-40分（3-4篇，每篇5题，每题2分）
4. 任务型阅读/补全对话：10分
5. 词汇运用/语法填空：10分
6. 句型转换/短文改错：5-10分
7. 书面表达：15-20分（80-120词）

### 4.4 教材版本适配规则
| 教材版本 | 适配要点 |
|----------|----------|
| 人教PEP（小学）/人教新目标（初中） | 话题贴合教材单元主题，词汇严格限定教材范围，题型常规，侧重基础句型运用 |
| 牛津译林版 | 阅读材料篇幅更长，任务型阅读占比提升，词汇量略大，侧重语境综合运用 |
| 外研版 | 话题覆盖更广，完形填空侧重逻辑推理，写作题型更贴近生活应用 |
| 沪教牛津版 | 整体难度偏高，词汇量更大，阅读包含拓展话题，写作要求更高 |

## 五、输出格式规范
```
### 5.1 卷头标准格式
# XX 年级英语 XX 考试试卷

（人教 XX 版）

满分：100 分  考试时长：60 分钟

## 注意事项

1. 答题前填写好自己的姓名、班级、考号等信息
2. 请将答案正确填写在答题卡上

### 5.2 题型排版规范
- 一级标题：听力部分、笔试部分
- 二级标题：具体题型名称，后标注总分，如「一、单项选择（共15小题，每小题1分，满分15分）」
- 小题用阿拉伯数字编号，选项A/B/C/D分行排列
- 阅读题先放原文，再放对应题目
- 作文题明确给出词数要求、话题提示与写作要点

### 5.3 答题卡规范
- 卷头预留姓名、班级、考号填写区
- 客观题：标准填涂方框，对应题号与A/B/C/D选项
- 主观题：给出对应答题横线，预留足够答题空间
- 作文题预留对应行数的方格/横线区域

### 5.4 答案解析规范
- 按题型顺序排列答案
- 客观题给出唯一标准答案
- 主观题给出参考范文与分层评分标准
- 每道题标注对应考点，难题附带详细解题思路与易错点提示

## 六、边界与限制
1. 听力部分仅提供听力材料与题目文本，不生成音频文件
2. 纯AI生成模式下，教材词汇为通用范围，精准同步教材单元需对接对应教材知识点库
3. PDF生成需本地安装对应Python依赖，Skill仅提供调用脚本与格式规范
4. 禁止用于商业用途，对接第三方题库时需遵守对应平台版权规则
```
